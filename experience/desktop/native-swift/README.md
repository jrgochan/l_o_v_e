## 🚀 Getting Started (Future)

*   Open `LoveSoul.xcworkspace` in Xcode 16+.
*   Select the `LoveApp` target.
*   Run (Cmd+R).

## 🛠 Prerequisites

*   **Xcode**: 16.0+ (Swift 6 support)
*   **macOS**: 15.0+ (Sequoia target)
*   **Hardware**: Apple Silicon (M1 or later required for ANE features)
