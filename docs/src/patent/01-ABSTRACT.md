# ABSTRACT

A computational system for modeling emotional states using a novel three-dimensional coordinate system comprising Valence, Arousal, and Connection (VAC) axes. The system converts emotional states into unit quaternions for smooth interpolation and 3D visualization. The Connection axis, measuring interpersonal alignment from -1 (separation/feeling FOR) to +1 (unity/feeling WITH), enables computational distinction between emotionally similar but relationally different states such as pity versus compassion. The system employs category-aware pathfinding with psychological constraints to generate therapeutically valid emotional transition paths, avoiding toxic positivity and respecting physiological arousal limitations. Multi-modal analysis compares voice prosody, semantic content, and self-reported emotions to detect incongruence. Machine learning personalizes regulation strategy recommendations based on individual history. The system provides real-time 3D visualization using SLERP interpolation between quaternion representations, enabling users to navigate their emotional landscape with evidence-based guidance. Applications include mental health therapy, emotional intelligence training, and human-computer interaction.

**Keywords:** emotional intelligence, quaternion mathematics, sentiment analysis, affective computing, mental health technology, emotion regulation, computational psychology, VAC model, SLERP interpolation, 3D visualization
