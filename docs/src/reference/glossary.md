## General Terms

| Term | Definition |
|------|-----------|
| **VAC** | Valence-Arousal-Connection coordinate system |
| **Valence** | Emotional positivity/negativity [-1, 1] |
| **Arousal** | Activation/energy level [-1, 1] |
| **Connection** | Interpersonal alignment [-1, 1] — the L.O.V.E. innovation |
| **Quaternion** | 4D number (w, x, y, z) used for smooth 3D rotations |
| **SLERP** | Spherical Linear Interpolation — smooth path between quaternions |
| **Soul Sphere** | 3D visual representation of an emotional state |
| **Atlas** | 87-emotion taxonomy based on Brené Brown's research |
| **Transition** | Path from one emotional state to another via therapeutic strategies |
| **Bridge Emotion** | Intermediate emotional state used to safely traverse emotional space |
