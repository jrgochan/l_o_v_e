## Quick Access

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/versor/calculate` | POST | Convert VAC coordinates to quaternion trajectory |
| `/versor/slerp` | POST | Compute SLERP interpolation between two quaternions |
| `/` | GET | Service info |
| `/health` | GET | Health check |

**Base URL:** `http://localhost:8001` (dev) | `https://love.jrgochan.io/versor` (prod)
