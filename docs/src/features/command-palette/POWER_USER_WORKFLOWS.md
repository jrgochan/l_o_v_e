### Workflow 1: The "Rapid Triage" (Clinician) 🔍

#### Scenario: Therapist wants to explore emotional neighborhood

#### Keyboard Sequence

```text
CMD+K → "joy"              (Find joy)
Enter                       (Select joy)
CMD+K → "~"                (Find similar - shows similar emotions)
↓ ↓ ↓ Enter                (Select peace)
⌘+Enter                    (Add to selection)
CMD+K → "~"                (Find more similar)
↓ ↓ Enter ⌘+Enter          (Add love)
CMD+K → "compute"          (Compute paths)
```

#### Time: ~15 seconds to explore emotional cluster

## Advanced Power User Tricks

### 1. Quick Toggles

#### Rapid view switching

```text
A      (Axes on/off)
S      (Sphere on/off)
L      (Labels on/off)
G      (Legend on/off)
F      (Focus mode on/off)
M      (Cycle animation modes)
```

**Chain without opening palette!**

### 2. Modifier Stacking

#### Multi-action selection

```text
CMD+K → "joy" → ⌘+Enter            (Add)
       → "peace" → ⌘+Enter          (Add)
       → "love" → ⌘+Enter           (Add)
       → "compute"                  (Compute paths for all)
```

### 3. Search Operators

#### Advanced filtering

```text
>expansive           (Category)
@favorite           (Favorites only)
~joy                (Similar to)
!joy                (Opposite of)
valence>0.5         (VAC filter)
arousal<0           (Low energy)
connection>0.8      (Highly connected)
```
