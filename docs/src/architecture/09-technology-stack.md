# Technology Stack

**Status**: Placeholder Stub
**Description**: Listing of all major technologies used in the L.O.V.E. platform.

## Backend (Python 3.11)

- **FastAPI**: High-performance async web framework.
- **Pydantic**: Data validation and hygiene.
- **Uvicorn**: ASGI server.
- **Faster-Whisper**: Local speech-to-text.
- **Spacy**: NER for PII scrubbing.

## Frontend (TypeScript)

- **Next.js 16**: React framework (App Router).
- **React Three Fiber (R3F)**: 3D rendering engine.
- **Three.js**: Core 3D library.
- **Zustand**: State management.
- **TailwindCSS**: Styling.

## Infrastructure

- **PostgreSQL 16**: Relational database.
- **Redis**: Message broker / Cache.
- **Ollama**: Local LLM inference server.
- **Podman / Docker**: Containerization runtime.
