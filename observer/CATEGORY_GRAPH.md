# Emotional Category Transition Graph

## Overview

This document defines the valid transitions between the 13 emotional categories in Brené Brown's "Atlas of the Heart". These rules ensure psychological validity in path planning.

## The 13 Categories

1. **Places We Go When Things Are Uncertain or Too Much**
2. **Places We Go When We Compare**
3. **Places We Go When Things Don't Go As Planned**
4. **Places We Go When It's Beyond Us**
5. **Places We Go When Things Aren't What They Seem**
6. **Places We Go When We're Hurting**
7. **Places We Go With Others**
8. **Places We Go When We Fall Short**
9. **Places We Go When We Search for Connection**
10. **Places We Go When The Heart Is Open**
11. **Places We Go When Life Is Good**
12. **Places We Go When We Feel Wronged**
13. **Places We Go To Self-Assess**

## Category Characteristics

### Category 1: Uncertain/Too Much (High Arousal, Stress)
- **Core Emotions**: Anxiety, Stress, Overwhelm, Fear, Panic, Dread
- **VAC Profile**: Negative valence (-0.3 to -0.7), High arousal (0.5 to 0.9), Negative connection (-0.2 to -0.5)
- **Psychological State**: Physiological activation, threat detection, survival mode
- **Key Feature**: Requires arousal regulation before complex processing

### Category 4: Beyond Us (Transcendent, Bridge)
- **Core Emotions**: Awe, Wonder, Curiosity, Surprise
- **VAC Profile**: Positive valence (0.5 to 0.7), Variable arousal, Positive connection (0.3 to 0.8)
- **Psychological State**: Openness, perspective-broadening, transcendence
- **Key Feature**: **GATEWAY CATEGORY** - Can reach most other categories

### Category 7: With Others (Connection, Healing)
- **Core Emotions**: Compassion, Empathy, Boundaries
- **VAC Profile**: Positive valence (0.2 to 0.5), Low-medium arousal, High positive connection (0.7 to 0.9)
- **Psychological State**: Relational attunement, feeling WITH others
- **Key Feature**: **HEALING CATEGORY** - Repairs shame, builds connection

### Category 8: Fall Short (Shame Core)
- **Core Emotions**: Shame, Guilt, Humiliation, Perfectionism
- **VAC Profile**: Very negative valence (-0.6 to -0.9), Low arousal (-0.1 to 0.2), Very negative connection (-0.6 to -1.0)
- **Psychological State**: Identity threat, isolation, unworthiness
- **Key Feature**: **MOST DIFFICULT TO EXIT** - Cannot skip to positive states directly

### Category 9: Search for Connection (Belonging Need)
- **Core Emotions**: Loneliness, Belonging, Insecurity, Fitting In
- **VAC Profile**: Variable valence, Low-medium arousal, Wide connection range (-0.9 to +1.0)
- **Psychological State**: Connection seeking, vulnerability
- **Key Feature**: **PIVOT CATEGORY** - Bridge between isolation and connection

### Category 11: Life Is Good (Goal States)
- **Core Emotions**: Joy, Happiness, Calm, Contentment, Gratitude, Tranquility
- **VAC Profile**: Positive valence (0.5 to 0.9), Variable arousal (-0.8 to 0.7), Positive connection (0.4 to 0.9)
- **Psychological State**: Well-being, safety, flourishing
- **Key Feature**: **DESTINATION CATEGORY** - Most common goal

## Critical Transition Rules

### Rule 1: Shame Cannot Skip to Joy
**Transition**: Category 8 → Category 11
**Difficulty**: 1.0 (Impossible)
**Reasoning**: Shame involves deep identity-level unworthiness and maximum disconnection. Joy requires connection and positive self-regard. These are psychologically incompatible without intermediate healing.

**Required Path**: Shame → Vulnerability (gateway) → Connection (Cat 7 or 9) → Joy
**Minimum Waypoints**: 2

**Example Emotions**:
- Shame [-0.9, -0.1, -1.0]
- → Vulnerability [0.0, 0.3, 0.6]  ← CRITICAL GATEWAY
- → Self-Compassion [0.6, -0.2, 0.9]
- → Joy [0.9, 0.7, 0.8]

### Rule 2: High Arousal Must Decrease Before Complex Cognition
**Transition**: Category 1 (high arousal) → Categories requiring cognitive work
**Reasoning**: Elevated arousal activates sympathetic nervous system, limiting prefrontal cortex function. Cannot do complex reappraisal or meaning-making while in fight-or-flight.

**Arousal Threshold**: If arousal > 0.6, must reduce to < 0.4 before:
- Category 4 (Wonder, Awe)
- Category 7 (Empathy, Compassion)
- Category 11 (Calm, Contentment)

**Strategies Required**: Physiological downregulation (breathing, movement, sensory)

### Rule 3: The Awe Bridge (Category 4 as Gateway)
**Special Property**: Category 4 emotions can be reached from almost any starting point.

**Why Awe Works**:
- Shifts perspective outside self (reduces rumination)
- Interrupts negative thought loops
- Creates sense of something larger (reduces shame)
- Can coexist with pain (bittersweet awe)

**Valid Transitions TO Category 4**:
- From Cat 1: Anxiety → Wonder (shift from threat to curiosity)
- From Cat 6: Grief → Awe (finding beauty in loss)
- From Cat 8: Shame → Awe (perspective shift, self-forgetting)

**Difficulty**: 0.3-0.5 (moderate, but effective)

### Rule 4: Connection Categories Are Healing
**Categories 7, 9, 10**: Connection-focused emotions

**Healing Property**: Positive connection directly counters:
- Shame (Cat 8): Connection proves worthiness
- Loneliness (Cat 9): Fulfills belonging need
- Anger (Cat 12): Connection enables perspective-taking

**Path Planning Principle**: When connection < 0, prioritize path through Cat 7/9/10

### Rule 5: Grief Is Different From Despair
**Grief** [Cat 6, -0.9, -0.4, +0.5]: Pain WITH connection
**Despair** [Cat 6, -0.9, -0.6, -0.7]: Pain PLUS disconnection

**Transition Rule**:
- Despair → other positive: MUST increase connection first
- Grief → acceptance: Can proceed naturally (connection already present)

**Clinical Note**: Don't "fix" grief. It's healthy pain. Path should be:
- Grief → Sadness → Acceptance → Peace
- NOT: Grief → "forced happiness"

### Rule 6: Comparative Emotions Lead to Shame
**Category 2** → **Category 8**: HIGH RISK path
**Reasoning**: Comparison often triggers inadequacy feelings

**Envy/Jealousy** → **Shame**: Common spiral
**Prevention**: Route through Category 13 (Self-Assessment) first
- Envy → Humility (recognize shared humanity)
- Humility → Self-Compassion
- Self-Compassion → Contentment

### Rule 7: Anger Requires Action or Release
**Category 12**: Action emotions (anger, contempt, etc.)

**Cannot directly to**: Category 11 (Life Is Good) without addressing the anger
**Difficulty**: 0.6

**Why**: Anger carries unreleased energy + sense of injustice

**Required Intermediate**:
- Physical release (Cat 12 → regulated state)
- OR Cognitive processing (Cat 12 → Cat 3: Acceptance of disappointment)
- OR Boundary-setting (Cat 12 → Cat 7: Assertiveness)

### Rule 8: Vulnerability Is The Gateway Emotion
**Vulnerability** [0.0, 0.3, 0.6]: Present in both Cat 1 and Cat 9

**Special Status**: Required waypoint for:
- Shame → Any positive connection
- Fear → Trust
- Loneliness → Belonging
- Perfectionism → Authenticity

**Brené Brown**: "Vulnerability is the birthplace of love, belonging, joy, courage, empathy, and creativity."

**Path Planning**: If start has connection < -0.3 AND goal has connection > 0.5:
- → Route MUST include Vulnerability as waypoint

## Bridge Emotions (Inter-Category Connectors)

### Primary Bridges
1. **Vulnerability** [0.0, 0.3, 0.6] - Cat 1 & 9
   - Connects: Shame ↔ Belonging
   - Property: Gateway to connection

2. **Awe** [0.7, 0.5, 0.8] - Cat 4
   - Connects: Almost anything ↔ Transcendence
   - Property: Perspective shift

3. **Compassion** [0.5, 0.2, 0.9] - Cat 7
   - Connects: Pain ↔ Healing
   - Property: Connection repair

4. **Acceptance** [0.3, -0.2, 0.4] - Cat 3 & 11
   - Connects: Disappointment ↔ Peace
   - Property: Release of resistance

### Secondary Bridges
5. **Curiosity** [0.5, 0.6, 0.3] - Cat 4
6. **Gratitude** [0.8, 0.3, 0.9] - Cat 11
7. **Humility** [0.6, -0.2, 0.8] - Cat 13

## Prohibited Direct Transitions

These transitions should NEVER be direct; always require intermediate waypoints:

1. **Cat 8 → Cat 11**: Shame → Joy (diff: 1.0)
2. **Cat 8 → Cat 10**: Shame → Love (diff: 0.7)
3. **Cat 6 (Despair) → Cat 11**: Despair → Joy (diff: 0.8+)
4. **High Arousal Cat 1 → High Connection Cat 10**: Panic → Trust (diff: 0.8)

## Implementation Guidelines

### For PathPlanner Algorithm

```python
def is_category_transition_valid(from_cat: str, to_cat: str) -> bool:
    """Check if direct category transition is allowed."""
    difficulty = CATEGORY_TRANSITIONS.get((from_cat, to_cat), 0.5)

    # Block impossible transitions
    if difficulty >= 0.9:
        return False

    # Allow difficult but possible transitions
    return True

def requires_bridge_emotion(path: List[Emotion]) -> bool:
    """Check if path requires specific bridge emotion."""
    start = path[0]
    end = path[-1]

    # Shame to positive connection REQUIRES vulnerability
    if start.category == "Places We Go When We Fall Short" and \
       end.vac_vector[2] > 0.5:
        has_vulnerability = any(
            abs(e.vac_vector[0]) < 0.1 and  # Neutral valence
            0.2 < e.vac_vector[1] < 0.4 and  # Medium arousal
            e.vac_vector[2] > 0.5  # Positive connection
            for e in path
        )
        if not has_vulnerability:
            return True  # Must add vulnerability

    return False
```

def get_category_specific_strategies(from_cat: str, to_cat: str) -> List[Strategy]:

## Clinical Considerations

### Warning Signs
- **Path too short**: Shame → Joy in 1 waypoint = psychologically invalid
- **Skipping arousal regulation**: Panic → Compassion without downregulation
- **Forced positivity**: Grief → Joy (respects the pain first)
- **Connection without vulnerability**: Loneliness → Belonging (vulnerability is the bridge)

### Success Indicators
- Path respects arousal curve (up then down, or steady decrease)
- Connection generally increases (or stays positive if starting positive)
- Includes appropriate bridge emotions
- Strategies match category needs
- Estimated time is realistic (hours to weeks, not minutes to hours for major shifts)
